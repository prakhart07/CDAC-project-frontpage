Centralized Education Database Management System

Centralized Education Database Management System Web App

Our aim is to make "one-student one-record" and educational record paperless and Pan-India availability and connectivity,we implemented this Project through microservices,we used J2EE for developement of back-end(API's) ReactJs technology as a front-end user interface,for database we used MySQL RDBMS. There are 3 major actors Admin-performs all the operations of the web app,Institute-which will performs operation related to institute i.e. selecting course,creating desk,etc.,Student-will read its indivisual data only.
Micro-services:we implemented it by deploying it in two parts first is Admin login and other one is the all remaining operations.
